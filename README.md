# Clipboard-Shortcut
Simple clipboard history function using shortcuts on iOS
-----------------------------------------[IMPORTANT!!!!!!] [SET UP]
Before activating the shortcut, create a new contact with the first name "Clipboard". After creation do not edit the contact again. 
---------------------------------------
Once you copy text that you wish to be added to your clipboard history, activate the shortcut and choose "Add" to append the text. "Select" allows you to view and select an item to be re-copied to your clipboard. "Edit" allows you to remove certain items from your clipboard history. "Clear" emptys the entire clipboard history

I recomend that this shortcut is easily accesible. Maybe use the action button or back tap. I have my triple power button click set to turn on assitive touch, then I eddited the singe tap to be run the clipboard shortcut. That way its only on the screen when I need it.
